# OsEasy-ToolBox

### 🚀 简单的介绍...

基于Python+Flet框架的   
简陋的噢易多媒体网络教室工具箱  

> 只能说勉强能用 哥们技术力不足💦💦   
> 但是能帮你做课上的靓仔还是可以的   
> 至少我自己的电脑课上是能用的😎

#### ✨工具箱目前支持功能✨
- 替换粘滞键为cmd脚本
- 删除键盘锁驱动&控屏
- 一键重启学生端 (貌似没什么用?)
- 恢复删除掉的备份文件
- 外部cmd守护进程
- 解除网络限制锁
- 打开噢易自带工具
- 挂起学生端主进程

### 🤔 在下载之前.....
工具箱内有一个启动输入动态密码环节   
动态密码为电脑的**时间+日期**   
🌰 举个例子:   
假设现在的时间是``2023/11/23 9:02``  
则动态密码为 ``090220231123``    
**若剩余输入密码次数归0**   
**将会触发蓝屏惩罚**   
> ~~可以把工具箱丢留给下一个人~~  
> ~~没准Ta就触发蓝屏惩罚了~~😉   

要是懒得输入密码  
或者是想要将工具箱分享给你的好同学们   
也可以下载无密码版本的  
> ~~只是可能会少一点坑人的乐趣~~ 🤫(咳咳)
